# Docker Compose Setup  

## **🚀 Quick Start**  
```bash
docker-compose up -d
```

## **⚙️ Configuration**  
- **SonarQube Port:** `9000`  
- **PostgreSQL Credentials:** `sonar/sonar`  
- **Data Persistence:** Docker volumes  

## **📜 Commands**  
| Command | Description |
|---------|-------------|
| `docker-compose up -d` | Start services |
| `docker-compose down` | Stop services |
| `docker-compose logs` | View logs |
